Instrucciones
Formato del Excel:

Tu archivo Excel debe tener las siguientes columnas:

  • Articulo: Nombre del producto (OBLIGATORIO)
  • Costo: Precio de costo (opcional)
  • Precio Venta: Precio de venta (opcional)
  • Codigo: Código de barras (opcional)
Comportamiento:
  • ✅ Solo necesitas el nombre del producto
  • ✅ Si no hay código de barras, se genera automáticamente
  • ✅ Si no hay precios, se crean con $0 (puedes editarlos después)
  • ✅ Si el código de barras existe, se actualiza el producto
  • ✅ Los productos se marcan como activos automáticamente
  • ⚠️ Se muestran errores si hay problemas en el archivo
